Diamond Command Usage.
=======================================

Usage:
[cmd][input]!
OR for comments:
[.][comment sentence]

Set variables A, B, C, D, E, F, G:
[a/b/c/d/e/f/g][input]!

Other:
[< (log)/> (input)/+/-/& (multiply)/( (square)/) (cube)/% (type)][input]!

Commands are:

< (log), > (input), + (add), - (minus), & (multiply), ( (square), ) (cube) and % (type - NUM or WRD).

=======================================

Setup.
=======================================

To setup, run

make -fbuild.mk
